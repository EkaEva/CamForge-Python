## 项目简介

CamForge 是一款基于 Python 的凸轮机构运动学仿真桌面应用，面向机械工程教学与设计场景。支持六种经典从动件运动规律、凸轮廓形解析计算、压力角实时可视化以及全程旋转动画，帮助工程师和学生直观理解凸轮机构的运动学特性。

### 核心亮点

- **六种运动规律** — 等速、等加速等减速、简谐、摆线、五次多项式、七次多项式，推程与回程可独立选择
- **解析计算** — 廓形坐标与压力角均采用解析公式，非数值微分，精度高、无跳动
- **实时动画** — 可配置帧数旋转动画，支持速度调节、暂停/继续/重播，同步显示压力角弧线与法线/切线
- **滚子从动件** — 支持滚子半径参数，自动计算实际廓形与理论廓形，干涉预警
- **现代化 UI** — CustomTkinter 框架，Apple/iOS 设计风格，卡片式布局，圆角按钮，拨动开关
- **多语言界面** — 支持中文、英文双语实时切换，导出文件名随语言自动变更
- **一键导出** — 静态图表导出为 600 DPI 高清 TIFF，动画导出为 150 DPI 循环 GIF，数据导出为 Excel/CSV，DXF 格式支持 CAD 软件导入
- **快速预设** — 内置 5 种常用凸轮预设（默认、小型、大型、高速、滚子），一键加载
- **批量导出** — 一键导出所有格式，无需逐个勾选
- **参数提示** — 鼠标悬停输入框显示参数合法范围
- **配置记忆** — 自动保存上次使用的参数、导出选项、UI 设置，下次启动自动恢复
- **键盘快捷键** — Enter 开始仿真、Space 暂停/继续、Left/Right 跳帧、R 随机凸轮
- **随机参数** — 一键生成合法随机凸轮参数，快速探索不同构型
- **参数校验** — 自动检测四角之和、偏距范围、滚子干涉等约束，异常参数即时提示

### 运动规律一览

| 编号 | 运动规律 | 推程位移公式 | 特性 |
|:----:|---------|------------|------|
| 1 | 等速运动 | $s = h \cdot \dfrac{\delta}{\delta_0}$ | 速度恒定，行程始末有刚性冲击 |
| 2 | 等加速等减速 | $s = 2h\left(\dfrac{\delta}{\delta_0}\right)^2$（前半段） | 加速度恒定，行程中点有柔性冲击 |
| 3 | 简谐运动 | $s = \dfrac{h}{2}\left(1 - \cos\dfrac{\pi\delta}{\delta_0}\right)$ | 无刚性冲击，行程始末有柔性冲击 |
| 4 | 摆线运动 | $s = h\left(\dfrac{\delta}{\delta_0} - \dfrac{1}{2\pi}\sin\dfrac{2\pi\delta}{\delta_0}\right)$ | 无刚性冲击和柔性冲击，动力性能好 |
| 5 | 五次多项式 | $s = h\left(10t^3 - 15t^4 + 6t^5\right)$ | 高阶平滑，综合性能优 |
| 6 | 七次多项式 | $s = h\left(35t^4 - 84t^5 + 70t^6 - 20t^7\right)$ | 最高阶平滑，边界速度加速度均为零 |

## 快速开始

### 环境要求

- Python 3.10 及以上
- Windows / macOS / Linux

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动应用

```bash
cd CamForge
python main.py
```

### 下载可执行文件

前往 [GitHub Releases](https://github.com/EkaEva/CamForge/releases) 下载最新版本的 Windows 可执行文件（无需安装 Python）。

## 使用指南

### 1. 参数输入

在左侧面板中依次填入凸轮设计参数：

| 参数 | 符号 | 说明 | 约束 |
|-----|:----:|------|------|
| 推程运动角 | $\delta_0$ | 推程阶段凸轮转角 | 整数，> 1° |
| 远休止角 | $\delta_{01}$ | 从动件停留在最远位置 | 整数，> 1° |
| 回程运动角 | $\delta_{0}'$ | 回程阶段凸轮转角 | 整数，> 1° |
| 近休止角 | $\delta_{02}$ | 从动件停留在最近位置 | 整数，> 1° |
| 行程 | $h$ | 从动件最大位移 | > 0 mm |
| 基圆半径 | $r_0$ | 凸轮基圆半径 | > 0 mm |
| 偏距 | $e$ | 从动件导路偏移量 | < $r_0$ mm |
| 角速度 | $\omega$ | 凸轮旋转角速度 | > 0 rad/s |
| 滚子半径 | $r_r$ | 滚子从动件半径 | < 最小曲率半径，0 为尖顶 |
| 离散点数 | — | 计算分辨率 | 36-3600，默认 360 |
| 压力角阈值 | — | 警告阈值 | > 0°，默认 30° |

> 四个角度之和必须恰好等于 **360°**。

### 2. 运行仿真

1. 点击 **「开始」** 按钮（或按 Enter），计算并绘制静态曲线与凸轮廓形
2. 动画自动开始播放，展示凸轮旋转过程
3. 使用 **「暂停」** 按钮暂停/继续动画（或按 Space）
4. 动画结束后按 Space 或点击 **「重播」** 重新播放
5. 拖动 **速度滑块** 调节动画帧率（1 = 最慢，10 = 最快）

### 3. 导出结果

勾选需要导出的内容，点击 **「下载」** 按钮：

| 类型 | 格式 | 说明 |
|------|------|------|
| 运动线图 | TIFF (600 DPI) | 推杆位移/速度/加速度三 Y 轴曲线 |
| 几何约束 | TIFF (600 DPI) | 压力角/曲率半径双 Y 轴曲线 |
| 廓形图 | TIFF (600 DPI) | 凸轮廓形 + 基圆 + 偏距圆 + 固定铰支座 |
| 动画 | GIF (150 DPI) | 360 帧循环动画，含进度提示 |
| CSV | CSV | 转角、向径、位移、速度、加速度、曲率半径、压力角 |
| Excel | XLSX | 同 CSV，7 列数据 |
| SVG | SVG | 矢量图：运动线图 + 几何约束 |
| DXF | DXF | 凸轮廓形 CAD 格式，支持 AutoCAD 导入 |
