## 二、最適化方向

### Phase 1 — 修正と補強（v0.2）

目標：機能の正確性とクロスプラットフォーム可用性に影響する重要な問題を解決。

- [x] **P1-1** `requirements.txt` を補完し、`Pillow` 依存関係を追加
- [x] **P1-2** クロスプラットフォームフォントの修正：実行時にシステムで利用可能な中国語フォントを検出し、自動フォールバック
- [x] **P1-3** クロスプラットフォームスクロールイベントの修正：Windows / macOS / Linux の `delta` 値とイベントタイプを区別
- [x] **P1-4** ウィンドウ最大化互換性の修正：プラットフォームに応じて `state('zoomed')` または `attributes('-fullscreen')` を選択
- [x] **P1-5** 浮動小数点検証問題の修正：`_read_params` で角度を先に `int` に変換してから `validate_params` に渡す
- [x] **P1-6** `LICENSE` ファイルの追加（MIT）
- [x] **P1-7** `.gitignore` の充実（IDE ディレクトリ、仮想環境、ビルド成果物など）
- [x] **P1-8** `cam_mechanics.py` に単体テストを追加（pytest、59 項目すべて通過）：
  - 5 種類の運動法則の境界値と連続性
  - カム輪郭の閉合性（始端と終端の座標が一致）
  - 圧力角計算の正確性
  - パラメータ検証が全分岐をカバー

### Phase 2 — リファクタリングと性能（v0.3）

目標：コード構造を改善し、アニメーション性能とエクスポート体験を向上。

- [ ] **P2-1** `CamSimulator` を独立モジュールに分割：
  ```
  ui/
  ├── sidebar.py      # サイドバー構築とパラメータ管理
  ├── toolbar.py      # ツールバーボタンとコントロール
  ├── plots.py        # 静止図表描画
  ├── animation.py    # アニメーション制御とフレームレンダリング
  ├── export.py       # 画像/GIF エクスポート
  └── app.py          # メインウィンドウの組み立てと起動
  ```
- [x] **P2-2** マジックナンバーを名前付き定数または設定項目に抽出：
  ```python
  TIP_WIDTH_RATIO = 0.04
  TIP_HEIGHT_RATIO = 0.08
  ROD_LENGTH_RATIO = 4.0
  ARC_RADIUS_RATIO = 0.3
  MAX_PRESSURE_ANGLE = 30.0
  ```
- [x] **P2-3** GIF エクスポートをバックグラウンドスレッド + プログレスバーに変更、UI フリーズを回避
- [x] **P2-4** アニメーションと GIF エクスポート間のプッシュロッド描画重複コードを削除し、共有関数に抽出
- [x] **P2-5** 静止図エクスポートを直接再計算による描画に変更（`ax.get_lines()` のコピーではなく）、`BlendedGenericTransform` ハックを排除
- [x] **P2-6** アニメーション終了後に「リプレイ」ボタンをサポート、再計算不要
- [x] **P2-7** ダウンロードファイル名に英語を使用（displacement.png / velocity.png / cam_profile.png / cam_animation.gif）

目標：シミュレーション機能を拡張し、エンジニアリング実用性を向上。

- [ ] **P3-1** ローラ従動件サポート：ローラ半径パラメータを追加し、実際の輪郭（等距離曲線）と理論輪郭を計算

## 四、優先度原則

1. **正確性優先** — 機能結果に影響する問題（依存関係欠落、浮動小数点検証）は体験問題より優先
2. **クロスプラットフォーム優先** — フォントとイベントの互換性は、macOS/Linux ユーザーが正常に利用できるかを直接左右
3. **テスト可能性優先** — 純粋計算モジュールにテストを追加することは、以降のすべてのリファクタリングの安全網
4. **漸進的リファクタリング** — 大きなクラスの分割時に機能を不変に保ち、各ステップが独立して検証可能
5. **ユーザー体感優先** — UI フリーズ、進捗フィードバックなしなど、使用体験に直接影響する問題は早期に解決
