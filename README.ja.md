# CamForge

**尖頂直動従動件盤形カム機構運動学シミュレーションシステム**

## プロジェクト概要

CamForge は、Python ベースのカム機構運動学シミュレーションデスクトップアプリケーションで、機械工学の教育と設計シーンに向けられています。5 つの古典的な従動件運動法則、カム輪郭の解析計算、圧力角のリアルタイム可視化、全行程回転アニメーションをサポートし、エンジニアと学生がカム機構の運動学特性を直感的に理解できるよう支援します。

### 主な特徴

- **5 種類の運動法則** — 等速、等加速等減速、単振動、サイクロイド、5 次多項式。上昇行程と戻り行程を独立して選択可能
- **解析計算** — 輪郭座標と圧力角はすべて解析公式を採用。数値微分ではなく、精度が高く跳動なし
- **リアルタイムアニメーション** — 360 フレーム回転アニメーション、速度調整・一時停止/再開対応、圧力角弧線と法線/接線の同期表示
- **ワンクリックエクスポート** — 静止図表は高解像度 PNG、アニメーションはループ GIF でエクスポート
- **ランダムパラメータ** — ワンクリックで正当なランダムカムパラメータを生成、異なる構成を素早く探索
- **パラメータ検証** — 四角の和、オフセット範囲などの制約を自動検出、異常パラメータを即時通知

### 運動法則一覧

| 番号 | 運動法則 | 上昇行程変位公式 | 特性 |
|:----:|---------|------------|------|
| 1 | 等速運動 | $s = h \cdot \dfrac{\delta}{\delta_0}$ | 速度一定、行程の始端と終端で剛性衝撃あり |
| 2 | 等加速等減速 | $s = 2h\left(\dfrac{\delta}{\delta_0}\right)^2$（前半区間） | 加速度一定、行程中点で柔衝撃あり |
| 3 | 単振動 | $s = \dfrac{h}{2}\left(1 - \cos\dfrac{\pi\delta}{\delta_0}\right)$ | 剛性衝撃なし、行程の始端と終端で柔衝撃あり |
| 4 | サイクロイド運動 | $s = h\left(\dfrac{\delta}{\delta_0} - \dfrac{1}{2\pi}\sin\dfrac{2\pi\delta}{\delta_0}\right)$ | 剛性衝撃・柔衝撃なし、動力性能良好 |
| 5 | 5 次多項式 | $s = h\left(10t^3 - 15t^4 + 6t^5\right)$ | 高次平滑、総合性能最優 |

### アニメーション特性

- **反転法**：カムが回転し、従動件は固定、実際の運動を直感的に表示
- **圧力角弧線**：接触点に圧力角弧線と注釈をリアルタイム描画
- **法線/接線**：輪郭導関数方向を解析計算し、接触点の法線と接線を正確に描画
- **位相区切り線**：放射状に上昇/遠休止/戻り/近休止の 4 段階を標示
- **情報パネル**：現在の回転角、圧力角、変位、行程、初期変位をリアルタイム表示

## 使用ガイド

### 1. パラメータ入力

左側パネルにカム設計パラメータを順に入力します：

| パラメータ | 記号 | 説明 | 制約 |
|-----|:----:|------|------|
| 上昇運動角 | $\delta_0$ | 上昇行程におけるカム回転角 | 整数、> 1° |
| 遠休止角 | $\delta_{01}$ | 従動件が最遠位置に留まる角度 | 整数、> 1° |
| 戻り運動角 | $\delta_{0}'$ | 戻り行程におけるカム回転角 | 整数、> 1° |
| 近休止角 | $\delta_{02}$ | 従動件が最近位置に留まる角度 | 整数、> 1° |
| 行程 | $h$ | 従動件の最大変位 | > 0 mm |
| 基礎円半径 | $r_0$ | カム基礎円半径 | > 0 mm |
| オフセット | $e$ | 従動件ガイドの偏り量 | < $r_0$ mm |
| 角速度 | $\omega$ | カム回転角速度 | > 0 rad/s |

> 4 つの角度の和は正確に **360°** でなければなりません。

### 2. 運動法則の選択

- **上昇行程法則**：ドロップダウンから 1〜5 号の運動法則を選択
- **戻り行程法則**：ドロップダウンから 1〜5 号の運動法則を選択
- **回転方向**：時計回り / 反時計回り
- **オフセット方向**：正オフセット / 負オフセット

### 3. シミュレーションの実行

1. **「開始」** ボタンをクリックし、静止曲線とカム輪郭を計算・描画
2. アニメーションが自動的に再生され、カム回転過程を表示
3. **「一時停止」** ボタンでアニメーションの一時停止/再開
4. **速度スライダー** をドラッグしてアニメーションフレームレートを調整（1 = 最遅、10 = 最速）

### 4. 結果のエクスポート

- エクスポートする図表タイプにチェック（変位 / 速度 / 加速度 / アニメーション）
- **「画像ダウンロード」** ボタンをクリック
- 静止図表は **PNG**（150 DPI）、アニメーションは **GIF**（ループ再生）で保存

### 5. ランダム探索

**「ランダムカム」** ボタンをクリックすると、制約を満たすランダムパラメータが自動生成され、異なるカム構成を素早く体験できます。

### 設計原則

- **計算と表示の分離**：`cam_mechanics.py` は純粋な数学ライブラリであり、GUI フレームワークに依存しないため、単独でバッチ計算や他プロジェクトへの統合が可能
- **解析優先**：すべての導関数と圧力角は解析公式（`ds/dδ = v/ω`）を採用し、数値微分による精度損失を回避
- **1° ステップ離散化**：全行程 360 個の離散点、各整数角度に対応する計算点を持ち、精度と性能を両立

## コアアルゴリズム

### カム輪郭計算（反転法）

輪郭座標は反転法により解析的に求められます：

$$
x = -s_n \left[ (s_0 + s)\sin\delta + p_z \cdot e\cos\delta \right]
$$

$$
y = (s_0 + s)\cos\delta - p_z \cdot e\sin\delta
$$

ここで $s_0 = \sqrt{r_0^2 - e^2}$、$s_n = \pm 1$ は回転方向符号、$p_z = \pm 1$ はオフセット符号です。

### 圧力角計算

$$
\alpha = \arctan\!\left(\frac{ds/d\delta - p_z \cdot e}{s_0 + s}\right)
$$

最大圧力角が **30°** を超える場合、インターフェースに赤色の警告が表示されます。

## よくある質問

<details>
<summary><b>起動後にインターフェースの中国語が四角で表示される</b></summary>

本プログラムは SimHei / Microsoft YaHei フォントで中国語をレンダリングします。システムにこれらのフォントがない場合：
- **Windows**：通常はプリインストール済み、対応不要
- **macOS**：SimHei フォントをインストールするか、`main.py` の `plt.rcParams['font.sans-serif']` をシステムに存在する中国語フォントに変更
- **Linux**：`sudo apt install fonts-wqy-microhei` を実行し、フォント設定を変更

</details>

<details>
<summary><b>アニメーション再生がカクつく</b></summary>

- 速度スライダーのレベルを下げる
- 不要な表示オプション（法線/接線、グリッドなど）をオフにする
- アニメーションは遅延を減らすため、デフォルトで 2 フレームごとにキャンバスを更新

</details>

<details>
<summary><b>GIF エクスポートに失敗する</b></summary>

Pillow がインストールされていることを確認してください：`pip install Pillow`。GIF エクスポートでは 360 フレームの画像を逐次レンダリングするため、時間がかかります。進捗通知が表示されるまでお待ちください。

</details>
